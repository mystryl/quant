# 回测报告 - SuperTrend_Enhanced 标准参数

**回测时间**: 2026-02-15 14:11

## 策略信息

**策略名称**: SuperTrend_Enhanced

**参数设置**:
- `period`: 10 - ATR周期
- `multiplier`: 3 - ATR倍数
- `n`: 3 - 突破确认系数
- `trailing_stop_rate`: 80 - 跟踪止损率（固定）

**策略描述**:
增强版SuperTrend指标的标准参数配置。

与SF14Re参数对比：
- SF14Re: period=50, multiplier=20（更保守，交易次数少）
- 标准参数: period=10, multiplier=3（更敏感，交易次数多）

**数据来源**: Qlib数据
**频率**: 15分钟和60分钟
