# 回测报告 - 标准SuperTrend 多参数测试

**回测时间**: 2026-02-15 09:00

## 策略信息

**策略名称**: 标准SuperTrend (非增强版)

**参数设置**:
测试了多组参数组合：
- `period=10, multiplier=3.0` - 标准参数
- `period=10, multiplier=2.0` - 较小倍数
- `period=7, multiplier=3.0` - 较短周期
- `period=14, multiplier=2.5` - 中等参数

**策略描述**:
标准SuperTrend指标（非增强版），基于ATR的趋势跟踪指标。

测试了不同参数组合在多个频率和年份上的表现，用于评估参数对策略性能的影响。

## 数据配置

**数据来源**: Qlib数据
**频率**: 15分钟和60分钟
**年份**: 2023, 2024, 2025
**合约**: RB9999.XSGE

**数据长度**:
- 2023年15分钟: 6,479根K线
- 2023年60分钟: 2,159根K线
- 2024年15分钟: 6,452根K线
- 2024年60分钟: 2,150根K线
- 2025年15分钟: 6,488根K线
- 2025年60分钟: 2,162根K线

## 回测配置

**初始资金**: 1,000,000 CNY
**交易手续费**: 0（假设）
**滑点**: 0（假设）

## 测试目的

通过多参数测试，评估：
1. 不同ATR周期对策略性能的影响
2. 不同ATR倍数对策略性能的影响
3. 不同频率（15分钟 vs 60分钟）的适用性
4. 参数稳定性（不同年份的表现）
